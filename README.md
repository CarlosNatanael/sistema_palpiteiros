# Sistema Palpiteiros

O **Sistema Palpiteiros** é uma aplicação web completa construída em Python (Flask) para gerenciar ligas e competições de palpites de futebol (Brasileirão, Copa do Brasil, Libertadores, etc.) entre amigos.

O objetivo é simples: reunir a galera, palpitar nos jogos da rodada e disputar o topo da tabela para ver quem é o verdadeiro mestre das previsões!
##  Tecnologias Utilizadas

* **Backend:** Python, Flask, Jinja2
* **Banco de Dados:** SQLite
* **Frontend:** HTML5, CSS3, JavaScript, Bootstrap 5
* **Arquitetura:** Separação limpa entre rotas (Backend), templates (Frontend) e chamadas de API externas.
